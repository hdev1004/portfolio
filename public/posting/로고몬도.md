# 로고몬도

설계장은 (주)티피엘엔디에서 제작한 B2B 형태의 도소매 사이트 입니다. <br/>

제작되어 있는 형태는 `Master`, `Master Admin` 으로 구분되어 있습니다. <br/>

프론트엔드 개발을 맡았고, React를 통해 개발을 진행했습니다. <br/> <br/>


## 기술 스택
---

`React`, `Framer Motion`, `Styled Compoment`
