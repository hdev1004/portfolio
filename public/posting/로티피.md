# LawTP

### 0. 프로젝트 소개


로티피는 티피엘엔디에서 제작했던 사내 법률 사무소 구축 프로젝트 입니다. <br/>

제작에 들어간 기술로는 `React`, `Styled Compoment`, `Framer Motion`을 사용했습니다. <br/>

짧은 개발 기간으로 만든 사이트이며, 반응형을 지원하고 있습니다. <br/>


<br/>
